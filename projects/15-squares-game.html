15SquaresGame

Java • Puzzle • Feb 27, 2023

Overview

A faithful implementation of the 15-puzzle in Java with attention to solvability and polished UI transitions. The shuffle employs a solvability-aware generator, and tile movement is animated for improved UX.

Key features

  • Solvable shuffle algorithm
  • Animated tile transitions and move counter
  • Time and score tracking for players

Tech stack

Java, Swing, OOP design patterns
