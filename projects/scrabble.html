Scrabble (Fork)

Java • Board Game

Overview

This project started as a fork of a community Scrabble implementation and includes my additions: robust scoring logic, tile bag shuffling, and clearer turn sequencing. I focused on correctness and maintainability of the core board logic.

Key features

  • Full scoring system including double/triple word/letter
  • Tile bag with frequency-controlled draws
  • Turn validation with basic AI opponent (optional)

Tech stack

Java, OOP design
