SoccerGame

Java • Game • Feb 28, 2023

Overview

SoccerGame is a Java-based soccer simulation featuring realistic ball physics, state-machine driven AI, and local multiplayer. I implemented the game loop, collision handling, and player controls. The project demonstrates systems design, vector math for physics, and event-driven UI.

Key features

  • Realistic ball physics using vector math and friction modeling
  • AI opponents with behavior states (seek, defend, shoot)
  • Local multiplayer and adjustable difficulty

Tech stack & notes

Java, Swing, custom game loop. See repository for build instructions and run scripts.
